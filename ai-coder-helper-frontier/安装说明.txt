依赖安装说明
================

由于 package.json 中新增了 marked 和 highlight.js 依赖，需要重新安装依赖包。

安装方法（任选其一）：

方法 1：使用批处理文件
--------------------
双击运行 install-deps.bat 文件

方法 2：使用命令行
--------------------
1. 在项目根目录下打开 CMD（命令提示符）或 PowerShell
2. 运行以下命令：
   npm install

方法 3：如果 npm 命令不可用
--------------------
1. 确保已安装 Node.js（从 https://nodejs.org 下载安装）
2. 安装完成后，重启命令行工具
3. 运行 npm install

安装完成后，重新启动开发服务器：
npm run dev
